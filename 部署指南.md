# 部署指南 — 让手机也能访问工作秘书

## 方法一：Vercel 一键部署（推荐，10分钟完成）

### 第一步：注册账号
1. 打开 https://vercel.com
2. 点击 "Sign Up"
3. 选择 **Continue with GitHub**（如无GitHub账号，先注册GitHub，也是免费的）

### 第二步：安装 Vercel CLI
打开终端，执行：
```bash
npm install -g vercel
```

### 第三步：部署
在终端执行：
```bash
cd /Users/laodaak/WorkBuddy/Claw/work-secretary
vercel
```
按提示操作：
- 选择 "Link to existing project?" → No
- 项目名称：work-secretary（直接回车）
- 部署目录：./ （直接回车）
- Override settings? → No

**部署完成后会获得一个网址，如：**
`https://work-secretary-xxx.vercel.app`

### 第四步：手机访问
1. 手机浏览器打开上面的网址
2. Safari（iOS）：点击底部分享按钮 → "添加到主屏幕"
3. Chrome（Android）：点击菜单 → "添加到主屏幕"
4. 之后桌面上出现"工作秘书"图标，像App一样打开！

## 方法二：GitHub Pages（完全免费，无需信用卡）

### 步骤
1. 注册 GitHub 账号：https://github.com
2. 新建仓库：New Repository → 名称：work-secretary → Public
3. 上传文件（把 work-secretary 文件夹里所有文件上传）
4. Settings → Pages → Source: main 分支 → Save
5. 等待约 1 分钟，访问：`https://你的用户名.github.io/work-secretary`

---

## 方法三：局域网访问（暂时使用）

如果只是在家里或公司内网使用，可以这样：

1. 打开终端，进入项目目录：
   ```bash
   cd /Users/laodaak/WorkBuddy/Claw/work-secretary
   python3 -m http.server 8080
   ```
2. 查看电脑的局域网 IP：
   ```bash
   ipconfig getifaddr en0
   ```
3. 手机连同一 WiFi，打开浏览器访问：
   `http://电脑IP:8080`

⚠️ 此方法只有电脑开着才能用，且只在同一WiFi下有效

## 配置 AI（获取 API Key 最简步骤）

### 推荐：通义千问（阿里云）
1. 打开 https://dashscope.console.aliyun.com/apiKey
2. 登录阿里云账号（可用支付宝扫码）
3. 点击"创建 API Key"
4. 复制 API Key
5. 在工作秘书中点击 ⚙️ → 填入 API Key → 保存

**免费额度**：新用户每月有免费额度，够个人办公使用
