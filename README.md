# 工作秘书平台 v1.0

> 纯前端工作助理，浏览器打开即用，数据自动保存

## 文件说明

| 文件 | 说明 |
|------|------|
| `index.html` | 主页面 |
| `style.css` | 样式文件 |
| `app.js` | 核心逻辑 |

## 快速开始

**第一步：打开平台**
- 双击 `index.html` 用浏览器打开
- 或右键 → 选择浏览器打开

**第二步：开始使用**
平台自动加载，无需安装任何软件

## 功能使用指南

### 📅 日程安排
1. 点击 **"+ 新增日程"** 按钮
2. 填写日程标题、开始/结束时间、类型、备注
3. 支持 **日/周/月** 三种视图切换
4. 日程自动按时间排序显示

### 📋 重点工作
1. 点击 **"+ 新增任务"** 按钮
2. 填写任务标题、所属项目、优先级、截止日期
3. 支持按优先级筛选（重要紧急 / 重要不紧急 / 已完成）
4. 点击"完成"按钮标记任务完成

**优先级说明：**
- 🔴 重要且紧急 — 立即处理
- 🟡 重要不紧急 — 安排计划
- 🟠 不重要但紧急 — 委托或快速处理
- ⚪ 普通 — 有空处理

### ✍️ 工作文本生成
1. 选择文本类型（通知/汇报/邮件/总结/计划）
2. 在输入框描述你的需求
3. 点击 **"✨ AI 生成"** 按钮
4. 生成后支持 **复制** 或 **下载**

> ⚠️ 当前为模拟生成，正式使用时可接入 OpenAI / 文心一言 等 API

### 📊 工作看板
- 自动统计本周工作完成情况
- 按项目分类展示任务分布
- 显示最近完成的任务列表
- 支持 **导出周报** 功能

## 数据管理

### 自动保存
- 所有数据自动保存在浏览器 `localStorage` 中
- 关闭页面后数据不丢失
- 再次打开页面自动恢复数据

### 备份导出
- 点击侧边栏 **"📦 导出备份"**
- 生成 `工作秘书备份_YYYY-MM-DD.json` 文件
- 建议每周备份一次

### 数据导入
- 点击侧边栏 **"📂 导入备份"**
- 选择之前导出的 `.json` 备份文件
- ⚠️ 导入会覆盖当前数据，请谨慎操作

## 常见问题

**Q：数据存在哪里？**
A：存在浏览器本地，清除浏览器数据会丢失，建议定期备份。

**Q：可以多人共用吗？**
A：当前版本为个人使用，数据存在本地。如需多人协作，需要部署后端服务。

**Q：AI 生成内容不准确？**
A：当前为模拟生成，接入真实 AI API 后可大幅提高质量。
